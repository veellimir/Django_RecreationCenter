# Django_RecreationCenter

<h3>Для разработки:</h3>

#### 1. Создать виртуальное окружения:
      python3 -m venv venv
#### Активировать:
      source venv/bin/activate
<hr>

#### 2. Установить Django &#128515;:
      pip3 install django
#### Запустить проект локально &#129300;:
      pip3 manage.py runserver
#### localhost:
      http://127.0.0.1:8000/
<hr>
